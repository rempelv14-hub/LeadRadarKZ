# LeadKZ Free v8

Эта версия бесплатная: без платного AI, без платных API и без авторассылки.

Главное:
- автопоиск валидных публичных групп Казахстана;
- проверка последних сообщений в группе;
- рейтинг групп 0–100;
- автоматический мониторинг только валидных групп;
- лиды по чат-ботам, CRM, Kaspi, заявкам, сайтам, SMM, дизайну;
- CRM, деньги, воронка, PDF-КП, кейсы;
- /check, диагностика, web-панель, автобэкап;
- запуск через START_HERE_WINDOWS.bat;
- Railway через TG_SESSION_STRING и Volume /data.
