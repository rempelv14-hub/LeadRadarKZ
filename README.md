
# LeadKZ Free v8 Easy Start + Railway

Версия v8 добавляет все 20 функций из последнего списка:

1. мастер настройки внутри Telegram (`/setup`);
2. команда `/check`;
3. панель диагностики;
4. логирование ошибок и продолжение работы после ошибок групп;
5. очередь “кому писать первым”;
6. красивые лид-карточки;
7. объяснение “почему это лид”;
8. объяснение “почему группа валидная”;
9. обучение фильтра 👍/👎 и командами `/learn_plus`, `/learn_minus`, `/learn_spam`, `/learn_geo`;
10. кнопка “не показывать похожее” через скрытие и минус-слова;
11. ежедневный отчёт;
12. недельный отчёт по плохим группам;
13. раздел “Доход” и команда `/money`;
14. шаблоны под разные ниши;
15. PDF-КП;
16. локальная Web-панель на `http://127.0.0.1:8080` или домене Railway;
17. multi-admin через `ADMIN_IDS`;
18. автобэкап базы в Telegram;
19. упрощённый Railway запуск через `START_HERE_WINDOWS.bat`;
20. режим “только Казахстан + только валидные бизнес-группы”.

## Самый простой запуск

На Windows запусти:

```bat
START_HERE_WINDOWS.bat
```

Выбери пункт `1`, введи BOT_TOKEN, ADMIN_ID, API_ID, API_HASH. Скрипт сам создаст `.env`, Telegram-сессию, `TG_SESSION_STRING`, `railway_variables.txt` и архив для деплоя.

## Railway

1. Создай проект Railway из GitHub.
2. Подключи Volume с mount path `/data`.
3. Перенеси переменные из `railway_variables.txt` в Railway → Variables.
4. Деплой запустится командой `python main.py`.

# LeadKZ Free v6 Auto Groups Kazakhstan

Бесплатный Telegram-бот для поиска клиентов по Казахстану. Версия v6 умеет сама искать публичные группы, проверять последние сообщения, оценивать валидность групп и мониторить только хорошие группы.

## Что нового в v6

- Автопоиск публичных KZ-групп по бизнес-запросам.
- Оценка группы 0–100 по гео, активности, лидам, спаму и рекламе исполнителей.
- Автостатусы групп: валидная, сильная, средняя, мусорная.
- Валидные публичные группы автоматически попадают в мониторинг.
- Мусорные группы можно скрывать автоматически.
- Режим `ONLY_VALID_GROUPS=true`: бот не шумит из плохих групп.
- Поддержка Railway через `railway.json`, `Procfile`, `TG_SESSION_STRING` и Railway Volume.
- Health endpoint для Railway: `GET /` возвращает статус.

## Быстрый запуск на ПК

```bash
pip install -r requirements.txt
copy .env.example .env
python create_session.py
python main.py
```

На Windows можно использовать:

```text
create_session_windows.bat
run_windows.bat
```

## Railway

Подробная инструкция есть в файле:

```text
RAILWAY_DEPLOY.md
```

Коротко:

1. Заполни `.env` локально минимум `API_ID` и `API_HASH`.
2. Выполни локально:

```bash
python create_session_string.py
```

3. Скопируй `TG_SESSION_STRING` из `session_string.txt`.
4. Загрузи проект на GitHub.
5. В Railway создай проект из GitHub repo.
6. В Variables добавь `BOT_TOKEN`, `ADMIN_ID`, `API_ID`, `API_HASH`, `TG_SESSION_STRING`.
7. Подключи Volume с mount path `/data` и поставь `DATA_DIR=/data`.

## Важно

Бот не делает авторассылку клиентам и не вступает в группы автоматически. Он анализирует только публично доступные группы и группы, доступные твоему Telegram-аккаунту. Это сделано для безопасности аккаунта.

# LeadKZ Free v6 Auto Groups Kazakhstan

Бесплатный Telegram-бот для поиска свежих лидов по Казахстану: чат-боты, CRM, автоматизация, сайты, SMM, дизайн, онлайн-школы, Kaspi-сценарии и запись клиентов.

## Что умеет v6

- поиск лидов за 1 час и 24 часа;
- гео-фильтр Казахстана: города, KZ, Kaspi, +7, тенге, ИП/ТОО;
- бесплатный скоринг 0–100 без платного AI;
- фильтр разработчиков, рекламы, спама и дублей;
- кнопки 👍/👎 для обучения фильтра;
- команды для добавления плюс-слов, минус-слов, спам-слов и гео-слов;
- очередь **кому писать первым**;
- отдельные разделы: **Kaspi-лиды**, **онлайн-школы**, **запись клиентов**;
- свежесть лида: до 1 часа, до 6 часов, 6–12 часов, 12+ часов;
- ниши: чат-боты, автоматизация/CRM, сайты, SMM, дизайн;
- оценка бюджета и риск ответа;
- CRM-статусы, избранное, архив;
- воронка продаж и учет денег;
- библиотека кейсов;
- рекомендации: что предложить клиенту, вопросы и мини-скрипт продажи;
- рейтинг групп и рекомендации плохих групп;
- поиск похожих групп по лучшим источникам лидов;
- готовые ответы, мини-КП и PDF-КП;
- заметки по лидам и напоминания;
- ежедневный план;
- рабочее время уведомлений и тихий режим;
- экспорт CSV/Excel, экспорт настроек, ручной бэкап и автобэкап SQLite.

## Безопасность

Бот не пишет клиентам автоматически, не вступает в группы автоматически и не делает массовые рассылки. Он только находит заявки и помогает тебе вручную ответить.

Доступ к меню разрешен только пользователю из `ADMIN_ID`. Файл `leadkz_session.session` никому не отправляй.

## Установка

```bash
pip install -r requirements.txt
copy .env.example .env
```

Заполни `.env`:

```env
BOT_TOKEN=токен_из_BotFather
ADMIN_ID=твой_telegram_id
API_ID=api_id_с_my.telegram.org
API_HASH=api_hash_с_my.telegram.org
TG_SESSION=leadkz_session
```

Первый раз создай сессию Telegram-аккаунта:

```bash
python create_session.py
```

Потом запускай:

```bash
python main.py
```

На Windows можно использовать:

```text
create_session_windows.bat
run_windows.bat
```

## Команды v6

Примеры:


## Важное про PDF

Для PDF-КП используется бесплатная библиотека `fpdf2`. Для русского текста нужен шрифт с кириллицей. На Linux обычно подходит DejaVuSans. На Windows бот попробует `Arial`/`Calibri`. Если шрифт не найден, бот отправит КП текстовым файлом.

## Если бот не видит сообщения

Полный мониторинг групп работает через `TG_SESSION`, то есть через аккаунт Telegram, который состоит в нужных группах. Обычный BotFather-бот сам по себе видит только ограниченный набор сообщений в группах.
